## Features

- OSRM 기반 실제 도로망 거리행렬
- ALNS 최적화 경로 탐색
- S3 결과 저장
- Athena 분석 연계
- ETA 시스템 연동

## Tech Stack

Python / AWS Lambda / SAM / MySQL / S3 / Athena / OSRM / ALNS
